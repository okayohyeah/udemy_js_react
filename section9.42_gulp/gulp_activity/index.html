Whose Who of GitHub

Whose Who of GitHub is a a static web page, where users can search for GitHub users by their usernames. Their search returns the following information about the user: name, username, avatar, location, as well as their followers and whom they are following. Avatars of all users are shown and links to their GitHub page.

RoverPix

RoverPix is a static site, which allows users to search for images of Mars from NASA's three Rovers and their various cameras by Martian sol day. Hover over an image to read its caption with details about the image. The page save the images until the user refreshes or leaves the page.
